# 《金色谱号：古典音乐闯关》玩法说明

## 一、游戏目标

玩家操控金色高音谱号，在限时跑酷中跳到正确答案的平台。游戏共 9 关，知识范围从早期巴洛克延伸至晚期浪漫主义，难度、倒计时压力和平台运动复杂度会逐关提高。

## 二、基本流程

1. 在主菜单选择“开始第一乐章”或进入关卡选择页。
2. 阅读画面上方的古典音乐知识题，每题有三个答案平台。
3. 先判断正确答案，并观察通向三个答案台的分支路线与平台运动预演。
4. 预演结束后点击“开始跑酷”，沿选定路线连续跳跃到答案平台。
5. 落台后查看答案分析、历史背景和记忆提示，再进入下一题。
6. 每关答完 3 道题即可结算；满足通关条件后解锁下一关。

## 三、操作方式

### 键盘

- 向左移动：`A` 或 `←`
- 向右移动：`D` 或 `→`
- 跳跃：`W`、`↑` 或空格
- 暂停或继续：`Esc`

### 手机和平板

使用舞台下方的“左移”“跳跃”“右移”触屏按钮。按钮支持按住连续移动。

## 四、判定与生命

- 跳到正确平台：计为正确答案，获得分数并显示知识解析。
- 跳到错误平台：扣除生命并显示正确答案与原因。
- 倒计时归零：视为本题失败并扣除生命。
- 撞到升号障碍：回到中央起点并扣除 3 秒，不扣生命，也不直接判错。
- 生命耗尽：本次闯关结束，可重新挑战。
- 答题、计分和通关判定均由本地规则完成，AI 不会修改正确答案。

## 五、九个关卡

1. 歌剧的黎明：蒙特威尔第、数字低音、单声歌曲、早期歌剧
2. 威尼斯的四季：维瓦尔第、科雷利、协奏曲、利托奈罗
3. 赋格与清唱剧：巴赫、亨德尔、赋格、组曲、清唱剧
4. 海顿的乐章：交响曲、弦乐四重奏、奏鸣曲式
5. 莫扎特的剧场：歌剧类型、钢琴协奏曲、古典主义均衡
6. 英雄的转调：贝多芬及古典主义向浪漫主义的过渡
7. 诗人与夜曲：舒伯特、肖邦、舒曼、浪漫主义小品
8. 幻想交响与乐剧：柏辽兹、李斯特、瓦格纳、威尔第
9. 世纪末的交响迷宫：勃拉姆斯、柴可夫斯基、德沃夏克、马勒、理查·施特劳斯

## 六、在线与离线题库

- 配置 DeepSeek API Key 后，DeepSeek 可动态出题并生成答案分析。
- 没有 Key、断网、请求超时或模型输出不合格时，游戏自动切换到 54 道内置离线题库。
- 离线模式具备完整的题目、答案、解析和通关流程，不影响正常游玩。

## 七、跑酷难度递进

- 第 1–2 关：每条路线包含 2 块宽踏板，用于熟悉连续跳跃和空中转向。
- 第 3–6 关：加入高低路线、窄踏板、水平或垂直移动平台及升号障碍。
- 第 7–9 关：加入交叉路线、按固定拍点显隐的平台和更短的判断时间。
- 路线难度会在 A、B、C 三个答案列之间轮换，不会让某个答案长期固定为最难路线。
- 每关时间依次为 34、32、30、28、27、26、25、24、23 秒。

## 八、关卡音乐

每关会显示当前演奏作品，并循环播放对应的浏览器合成主题编曲：

1. 《奥菲欧》托卡塔
2. 《四季·春》
3. 《C大调前奏曲》BWV 846
4. 《惊愕交响曲》第二乐章
5. 《小夜曲》K.525
6. 《第五交响曲》第一乐章
7. 《降E大调夜曲》Op.9 No.2
8. 《女武神的骑行》
9. 《自新大陆交响曲》第二乐章

音乐和效果音均可在“设置”中调节。答案解析时音乐会自动降低，暂停或退出关卡时同步暂停或停止。

## 九、通关提示

- 先读完三个平台的文字，再开始跑酷。
- 动态平台会先展示一个完整运动周期，此时倒计时不会减少。
- 所有路线踏板均为单向平台：上升时可以从底部穿过，下降时才会落在平台上。
- 高难度关卡中应先判断正确答案，再观察平台节奏，避免在空中临时改变目标。
